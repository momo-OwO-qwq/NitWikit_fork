## 支持方式

### 手机运行 Java

目前基本上有两种主流方案：Amethyst launcher 和 Fold Craft Launcher，其中后者对于中国用户更加友好，推荐使用

### Geyser 转换

使基岩版客户端进入 Java 版服务器的代理软件。

[点此阅读](/java/process/mobile-player/geyser/intro) 关于 Geyser 的文档。
